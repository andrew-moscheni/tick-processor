# Strategy Backtester

A sandbox environment to backtest the newest trading strategies
in quantitative finance. Gives a description of the algorithm 
being implemented in a IPYNB file, to then be implemented in 
practice once the strategy has been backtested to simulate putting 
a strategy into production. Built as a portfolio project demonstrating
Python, backtesting, algorithmic development, and data pipelines. 

## Features

- Algorithms first backtested and implemented as notebook files
- Once successfully backtested, algorithms are then written as
Python scripts, to then be used by ```main.py```
- ```main.py``` takes in a specific algorithmic strategy originally,
then asks for the specifics to the algorithm to simplify and modularize
algorithm implementation
